
# MD에서 폴더트리 구조 만드는법

1. VSCode에서 file-tree-generator 플러그인을 설치
2. 트리구조를 만들 폴더를 오른쪽 클릭하고 Generate to Tree 클릭
3. 완성

```markdown
📦public
 ┣ 📂fonts
 ┃ ┣ 📜Inter-italic.latin.var.woff2
 ┃ ┗ 📜Inter-roman.latin.var.woff2
 ┣ 📂images
 ┃ ┣ 📜photo.jpg
 ┃ ┗ 📜photo2.jpg
 ┣ 📂img
 ┃ ┗ 📂OOP
 ┃ ┃ ┣ 📜Untitled 1.png
 ┃ ┃ ┣ 📜Untitled 2.png
 ┃ ┃ ┗ 📜Untitled.png
 ┗ 📜favicon.ico
```
