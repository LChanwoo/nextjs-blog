# 대문자와 소문자

### **문제 설명**

문자열 `my_string`이 매개변수로 주어질 때, 대문자는 소문자로 소문자는 대문자로 변환한 문자열을 return하도록 solution 함수를 완성해주세요.

---

### 제한사항

- 1 ≤ `my_string`의 길이 ≤ 1,000
- `my_string`은 영어 대문자와 소문자로만 구성되어 있습니다.

---

### 입출력 예

| my_string | result |
| --- | --- |
| "cccCCC" | "CCCccc" |
| "abCdEfghIJ" | "ABcDeFGHij" |

---

### 입출력 예 설명

입출력 예 #1

- 소문자는 대문자로 대문자는 소문자로 바꾼 "CCCccc"를 return합니다.

입출력 예 #2

- 소문자는 대문자로 대문자는 소문자로 바꾼 "ABcDeFGHij"를 return합니다.

---

### 내 해답

```jsx
function solution(my_string) {
    var answer = '';
    for(let i=0;i<=my_string.length;i++){
        if(my_string.charAt(i)>='a' && my_string.charAt(i)<='z'){
            answer += my_string.charAt(i).toUpperCase() 
        }else if(my_string.charAt(i)>='A' && my_string.charAt(i)<='Z'){
            answer += my_string.charAt(i).toLowerCase() 
        }
    }

    return answer;
}
```
