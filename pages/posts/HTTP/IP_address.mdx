# ip주소의 구조

## 개요

인터넷 프로토콜 주소(Internet Protocol Address, IP 주소)는 컴퓨터 네트워크에 연결된 장치를 식별하기 위해 사용됩니다. 이 글에서는 IP 주소의 용도, 구조, 방식에 대해 살펴보겠습니다.

## IP 주소의 용도

IP 주소는 인터넷 상에서 장치를 식별하기 위해 사용됩니다. 여러 개의 컴퓨터가 연결된 네트워크 상에서 목적지 IP 주소를 가진 데이터가 보내질 때, 패킷은 목적지 컴퓨터로 보내집니다.

## IP 주소의 구조

IP 주소는 32비트 길이의 네트워크 주소로 구성됩니다. 네트워크 주소는 사용자가 장치 내부에 설정해야 하는 논리 주소와는 다릅니다. 네트워크 주소는 컴퓨터 네트워크 상에서 장치를 식별합니다.

## IP 주소의 방식

IP 주소는 현재 두 가지 방식으로 사용됩니다. 첫째는 IPv4 주소 형식입니다. IPv4 주소는 32비트 네트워크 주소로 구성되며, 4개의 8비트 블록으로 나뉘어져 있습니다. 둘째는 IPv6 주소 형식입니다. IPv6 주소는 128비트 네트워크 주소로 구성되며, 8개의 16비트 블록으로 나뉘어져 있습니다.

## 결론

IP 주소는 인터넷에서 장치를 식별하기 위해 사용되는 네트워크 주소입니다. 두 가지 방식의 IP 주소가 있으며, 각각 IPv4 주소와 IPv6 주소 형식으로 구성되어 있습니다. IP 주소는 컴퓨터 네트워크 상에서 장치를 식별하는 중요한 역할을 합니다.
