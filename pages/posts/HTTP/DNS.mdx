
# DNS

- DNS는 ‘Domain Name System(도메인 이름 시스템)’의 약자로, 인터넷에 연결된 장치를 식별하고 찾기 위해 사용되는 시스템입니다. 인터넷 주소의 종류로는 IP 주소와 DNS 주소가 있습니다. IP 주소는 컴퓨터에 사용되는 숫자로 구성된 주소로, DNS 주소는 사람이 이해하기 쉬운 문자로 구성된 주소로 컴퓨터를 식별하는 데 사용됩니다.
- DNS는 IP 주소와 DNS 주소를 매핑하기 위해 사용되며, 네트워크 상에서 인터넷 상의 호스트를 찾기 위해 사용됩니다. DNS 서버는 도메인 이름과 IP 주소를 관리하고, 사용자의 요청에 따라 IP 주소를 도메인 이름에 매핑합니다.
