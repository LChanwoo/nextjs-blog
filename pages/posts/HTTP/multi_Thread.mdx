
# 멀티스레드

 멀티스레드는 여러 작업을 동시에 실행하고 이를 병렬로 실행하는 것을 뜻합니다. 이것은 컴퓨터가 작업을 더 빠르게 처리하는 것을 도와줍니다.

### 멀티 스레드와 멀티 프로세스의 차이

멀티 스레드와 멀티 프로세스는 모두 두 개 이상의 작업을 병렬로 실행하는 것을 뜻합니다. 하지만 차이점은 여기에 있습니다:

- 멀티 스레드는 하나의 프로세스 내에서 여러 작업을 병렬로 실행합니다. 그래서 같은 메모리와 같은 자원을 공유합니다.
- 멀티 프로세스는 여러 개의 프로세스를 병렬로 실행합니다. 각 프로세스는 독립적이고 별도의 자원을 가집니다.
